Возможности платформы ПрофДНК

Короткий обзор функций, которые уже доступны в MVP для демонстрации на хакатоне.

Для администратора

  • Создание аккаунтов психологов (саморегистрация отключена).
  • Управление доступом: дата окончания доступа и блокировка.
  • Ручное продление доступа без автопродления.
  • Просмотр списка специалистов и оперативное изменение статуса.

Для психолога

  • Личный кабинет с профилем, фото и Markdown-блоком «О себе».
  • Публичная визитка и QR-код для клиентов.
  • Конструктор тестов: ручной сборщик + импорт/экспорт JSON.
  • Формулы метрик: настраиваемые вычисления производных показателей.
  • Именованные ссылки кампаний и группировка результатов по источникам.

Для клиента

  • Прохождение теста по уникальной ссылке без регистрации.
  • Обязательные поля клиента по настройкам методики.
  • Онлайн-прогресс во время заполнения опросника.
  • Генерация клиентского отчета (HTML/DOCX), если включено психологом.

Технически

  • Backend: FastAPI + SQLAlchemy.
  • База: PostgreSQL (локально и на VPS), деплой через systemd/nginx.
  • Отчеты генерируются по запросу и не хранятся файлами на диске.
  • Автодеплой на VPS через GitHub Actions при push в main.
  • Интерфейс адаптирован под desktop и mobile режимы.
