# Инструкция для психолога

Дата обновления: 2026-03-22

## Быстрый сценарий
1. Войти в кабинет по данным от администратора.
2. Создать методику в конструкторе (`/tests/new`).
3. Опубликовать тест и открыть его в разделе «Опросники» (`/tests`).
4. Сгенерировать ссылку для клиента или кампании.
5. Получить прохождения и сформировать отчёты (HTML/DOCX).

## Как создать методику
1. Откройте «Конструктор».
2. Заполните шаг «Основное»:
   - название;
   - описание;
   - обязательные поля клиента.
3. На шаге «Поля клиента» при необходимости добавьте доп. поля.
4. На шаге «Секции и вопросы» соберите структуру методики.
5. На шаге «Логика и формулы» задайте расчёт производных метрик.
6. На шаге «Публикация» нажмите «Создать тест».

## Как работать со ссылками
1. Откройте карточку теста.
2. Используйте:
   - основную ссылку для общего потока;
   - именованные ссылки для отдельных групп/кампаний.
3. Для ссылок можно задавать:
   - лимит прохождений;
   - срок действия;
   - персональную привязку по ФИО.

## Как смотреть результаты
1. В карточке теста смотрите таблицу прохождений.
2. Фильтруйте данные по кампаниям.
3. Открывайте клиентский и профессиональный отчёты.
4. Используйте экспорт (CSV/XLSX/PDF), если нужно передать данные.

## Возможности платформы
- Конструктор с секциями, типами вопросов и формулами.
- Ветвления вопросов/секций по условиям.
- Именованные кампании и аналитика по источникам.
- Публичная визитка психолога с QR-кодом.
- Отправка писем (если SMTP включён).

- В кабинете психолога: кнопка «Инструкция»
